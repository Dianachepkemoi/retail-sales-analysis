# Retail Sales Analysis

This project analyzes retail sales data and visualizes trends over time.

## 📁 Folder Structure

- `Notebook/analysis.py`: Main Python script for data analysis and visualization
- `data/retail_sales.csv`: Dataset used for analysis
- `visualizations/`: Output graphs saved as PNGs
